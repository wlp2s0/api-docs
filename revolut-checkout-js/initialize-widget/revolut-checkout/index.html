RevolutCheckout

Load the embed script and create an instance for the order token.

  • Production
  • Sandbox
type RevolutCheckout = (  `public_id`: string,  mode?: 'prod' | 'sandbox' = 'prod') => Promise<Instance>;
type RevolutCheckout = (  `public_id`: string,  mode?: 'prod' | 'sandbox' = 'sandbox') => Promise<Instance>;

Parameters#

Initialze the widget using an order token (also called public_id) and declaring the environment that you want to use:

  • Production
  • Sandbox
RevolutCheckout("<PUBLIC_ID>", "prod").then(function (instance) {  // Work with instance:  // instance.payWithPopup()  // instance.createCardField()  // ...});
RevolutCheckout("<PUBLIC_ID>", "sandbox").then(function (instance) {  // Work with instance:  // instance.payWithPopup()  // instance.createCardField()  // ...});
  • <PUBLIC_ID> is the public_id passed as the parameter to call the widget instance. You can find the value of public_id in the response that the Merchant API returns when you create an order successfully.
  • mode is the Merchant API environment, which can be either prod or sandbox. By default, this is set to prod. This is an optional parameter.

Returns Promise with an order Instance.

tip

The environment that you use must match the environment you used to create the order you want to make a payment for.
