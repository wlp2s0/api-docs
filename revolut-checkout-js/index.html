Introduction

The Revolut Widget allows you to take payments for the orders that you create via the API.

It's a very powerful tool that has many benefits:

  • Makes it very simple to initialize the different payment methods that Revolut offers without having to go through complex implementations.
  • It's initiated via an iframe. This makes it much easier as a merchant to stay PCI compliant.
  • Orchestrates any calls that are needed to the back-end to make sure that all the different cases that your customer could go through are handled.
  • Will always stay up-to-date with the latest improvements to any payment method that has already been implemented.
