HTTP Response errors

The API uses the following error codes:

CODEMEANING
400Bad Request -- Your request is invalid. Ensure that all the parameters provided in the request follow the API Reference.
401Unauthorized -- Your access token is wrong.
403Forbidden -- Access to the requested resource or action is forbidden.
404Not Found -- The requested resource could not be found.
405Method Not Allowed -- You tried to access an endpoint with an invalid method.
406Not Acceptable -- You requested a format that isn't JSON.
429Too Many Requests -- You're sending too many requests.
500Internal Server Error -- We had a problem with our server. Try again later.
503Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
